Список горячих клавиш для дневника по умолчанию

### Виджет открытой задачи

| Hotkey   | Описание                        |
| -------- | ------------------------------- |
| `Alt+w`  | Скрыть/Показать список подзадач |
| `Alt+n`  | Создать новую подзадачу         |
| `Ctrl+s` | Сохранить текущую задачу        |
