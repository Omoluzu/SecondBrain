`Этап1`
Возможность проставлять теги 

Необходимо место куда будут заводиться теки. 
Для этого необходимо продумать средство хранения тегов в базе данных

#### Этап 1.
- [ ] Описание структуры хранения тегов в БД
- [ ] Реализация хранения тегов в БД.
- [ ] Реализация модального окна для добавления тегов.
- [ ] Доработка модального окна редактирования задачи для возможности линковки к ней тегов.
- [ ] Вывод тегов в основной задачи

#### Этап 2.
- [ ] Модальное окно тегов.
	- [ ] Возможность редактирование наименования тегов
	- [ ] Возможность удаления тегов
- [ ] Задачи.
	- [ ] Возможность удаления тега с задачи.
