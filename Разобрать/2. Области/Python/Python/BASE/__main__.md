


#__main__ - чаще всегод используеться для представления интерфейча командной строки для поката.

bandclass  
  ├── __init__.py  
  ├── __main__.py  
  └── student.py  
  
  
#__main__ будет выполняться при вызове самого пакета непосредственно из командной строки с помощью флага `-m` 

```bash
python3 -m bandclass
```
