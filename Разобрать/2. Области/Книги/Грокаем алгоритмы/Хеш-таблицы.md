**Хеш-таблица** - это  [[Cтруктура данных]]  реализующая объединение [хеш-функции](Хеш-функция.md) и [массива](Массив.md)

**Быстродействие:**

| Действие | средний случай | худший случай |
| -------- | -------------- | ------------- |
| Чтение   | ==О(1)==       | ==O(n)==      |
| Вставка  | ==O(1)==       | ==O(n)==      |
| Удаление | ==O(1)==       | ==O(n)==      | 


В [[Python]] хеш-таблицы реализованы через работу с [[dict]] 
```python
book = dict()
book["apple"] = 0.67
book["milk"] = 1.49
book["avocado"] = 1.49

print(book["avocado"])
>> 1.49
```
